# Snail Lab

A Super Snail companion suite.

## Current structure

- `index.html` — Snail Lab homepage
- `styles.css` — shared styling
- `app.js` — shared navigation behavior
- `tools/biozilla/index.html` — Biozilla module placeholder
- `tools/apostle-analytics/index.html` — Apostle Analytics module placeholder

## GitHub Pages setup

1. Upload these files to your `snail-lab` repository.
2. Go to **Settings → Pages**.
3. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
4. Save.

Your site will be available at:

`https://YOUR-GITHUB-USERNAME.github.io/snail-lab/`
